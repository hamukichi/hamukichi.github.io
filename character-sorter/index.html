Character Sorter

入力した語句を構成する文字をソート（整列）します．ソートなぞなぞやマージなぞなぞの作問に使えます．

使い方

語句とその数を入力し，「ソート」ボタンを押してください． 複数の語句を使う場合（マージなぞなぞ）にも，そのまま続けて入力してください． するとその語句を構成する文字をつなげて，辞書式昇順（正確には UTF-16 のコードの昇順）に並べたものが出力されます．

「ソート」を押下する前に，「空白を除去する」をチェックしておくと，入力した語句中の空白が除去されてからソートが実行されます． マージなぞなぞの際に有用です．

ソートを実行後に，「ツイート」をクリックすると，ソートなぞなぞ（1 語の場合）またはマージなぞなぞ（2 語以上の場合）の問題形式で投稿できます．

注意点

  • ブラウザの JavaScript を有効にしてください．
  • 動作確認は Windows 10 上の Google Chrome で行いました．
    • 環境によっては正しく動作しないおそれがあります．
  • 入力された語句が，一般に通用するものであるとは限りません．
    • 現在辞書によるチェック機能の実装を検討中です．
